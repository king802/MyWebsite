Data Science and AI Projects

The Data Science/Artificial Intelligence field of computer science is fascinating to me. The challenge of working with large datasets to manipulate and extract information is something that I have come to enjoy. In this section, I have outlined some of the projects I have done in the data science field.

NLP for Genome sequencing

In this project, I got my first exposure to NLP. The goal of this project was to extract information from clinical text documents and use it to classify the gene mutation in 1 of 9 categories. Since this was my first time working with text documents and extracting information, I utilized the SciKit Learn python library and used its tokenization tools to classify the clinical text.

In the future, I hope to revisit this project to try and improve upon my classifier with some of the new techniques that I have learned. I think that using a customized tokenization tool built with medical literature in mind would help significantly.

Quote Retrival Algorithm

In this project, I worked in a team of 3 to develop a search engine that would return relevant documents, specifically quotes and opinions related to the query. To complete this project, I used text processing techniques such as tokenization and stemming to engineer a basic search engine in Jupyter Notebook.

Chicago Crime Data Exploration

I was able to put some of my data mining techniques together in this project by implemented various preprocessing procedures such as dimension reduction, feature selection and label encoding to clean up raw datasets from Kaggle. I also used the Apriori technique to try and identify what features correlated strongly together.

The lessons learned from this project were structured to view and manipulate the data and not about getting meaning information from the data. I hope to be able to come back to this data set in the future to try and do some more digging to draw better conclusions from the data.

NLP to classify toxic Text.

This project was part of an Information Retrieval course that I took at Northeastern. In this python code, I implemented a naive Bayes text categorization system that would predict whether a movie review was positive or negative. Implementing the NLP techniques and naive Bayes classifier from scratch was very interesting, and in doing so, I was able to play with the implementation to see how using different n-grams, and other smoothing techniques would affect the outcome of the prediction.

I did this project in plain python, but I would like to try to come back to this project and use a Jupiter notebook. It would benefit the project and allow me to show some more interesting differences in the classifier's tweaking.

Pacman AI Agents Project

This trendy project I worked on while completing my Fundamentals of AI course at Northeastern. It was based off of a popular UC Berkeley course that uses Pac Man to implement AI agents. In this project, I programmed different algorithms such as reinforcement Learning, A* as well as other Baysian based agents in python.

Full Stack Java Chat Application

I worked on this project in a team of 4, working in an agile work environment. The goal of this project was to build out a full stack chat application. Cap Stone Project

This was a project that was completed as a Capstone. Its had three main goals. The first to upgrade and fix the small engine dynamometer at WIT. The second goal was to be able to use the computer software EES (Engineering Equation Solver) to successfully be able to predict the emission trends of the engine on the the test stand. Lastly the project was to be converted into a lab so that future engineering students at Wentworth could use the engine test stand and the simulation to learn.

As one of four group members on this project my focus was mainly on setting up the testing procedure for the test stand. I then had to mimicking that procedure on EES to try to correlate the data. This involved manipulating the test environment and engine set points (engine speed, torque, etc) to be ale to make correct assumptions in the EES program. Along with this work I also assisted team members in the refurbishing of the engine test stand which included adding and calibrating sensors.

This project resulted in a formal presentation, report and a written test procedure for how to operate the test stand and simulation. The team was able to work together successfully to accomplish all of its goals and the project was a success that resulted in a high grade.
